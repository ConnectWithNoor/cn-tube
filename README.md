## Tech Stack
The application is following the opinionated airbnb linting as well as the style guide. 

**Language:** TypeScript

**Development Tool:** Vite

**Client:** React, ReduxToolkit, TailwindCSS

**Server:** Node

**Lint & Style Guide:** Airbnb-Eslint, Prettier
